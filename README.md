## Features

- Modern design with smooth animations and transitions
- Dark and light mode toggle with persistent preference storage
- Fully responsive layout optimized for all screen sizes
- Interactive project carousel with dynamic navigation
- Contact form integrated with EmailJS for message delivery
- Scroll-triggered reveal animations
- Unique visual identity for each section
- Custom styled accordion-based about section
- Project screenshots with overlay effects

## Technologies Used

- HTML5 - Semantic markup structure
- CSS3 - Custom properties, Flexbox, CSS Grid
- JavaScript - ES6+ with DOM manipulation
- Font Awesome - Icon library
- Google Fonts - Space Grotesk, Inter, JetBrains Mono
- EmailJS - Contact form functionality
